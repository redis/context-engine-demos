# ElectroHub Director Demo

Technical Reveal.js presentation for the ElectroHub domain pack.

Open locally:

```bash
cd domains/electrohub/presentations/director-demo
python3 -m http.server 8015
```

Then browse to:

```text
http://127.0.0.1:8015
```

The deck is self-contained except for Reveal.js assets loaded from the public CDN.
