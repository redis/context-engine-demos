# ShiftIQ Engineering Brief

Technical Reveal.js presentation for the ShiftIQ finance-researcher domain pack.

Open locally:

```bash
cd domains/finance-researcher/presentations/engineering-brief
python3 -m http.server 8016
```

Then browse to:

```text
http://127.0.0.1:8016
```

The deck is self-contained except for Reveal.js assets loaded from the public CDN.
